Settings

General

Use this as your admin control area. Update master prices, categories, presets, and quick-estimate values in Google Sheets, then open them from here. The default terms here are copied into a project when you click Load Default Terms in Project Setup.

Company Information

These details replace the ModuQuote name on printable forms. Quotation, BOQ, costing sheets, and acknowledgement receipts will use this header automatically.

Pricing

Client Item Pricing

When enabled, the hidden project-level costs are distributed to the item rows so the client's item amounts equal the final project total.

Budgetary Rates

Fixed Cost shows one number per item and total. Budgetary Range shows a low-to-high range for each item and for the project total.

Presets

Edit your categories and preset packages here. When a category is selected in Cost Builder, matching preset materials and accessories can auto-load into a new empty item.

Clause Library

Format: Title::Clause text, separated by --- on its own line. These clauses appear in Project Setup so you can insert them into the quotation terms.

Data Source

This build is integration-ready. Add your Apps Script URL, then the app can pull master data and push or pull projects. The endpoint should return JSON and accept project payloads.

Print Preferences

Use these to tighten the print layout for quotations, BOQ, and internal costing sheets. Client quotations now include Qty, Unit Price, and Amount.

About This Settings Panel
